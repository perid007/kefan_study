# 古诗闯关（统编版·三四年级）

一个面向小学生的古诗学习网页应用。支持逐首闯关、家长检查与参考答案查看，帮助孩子“能背会懂”后再进入下一首。

## 主要功能
- 逐首闯关与解锁（按年级与上下册顺序）
- 模块化学习
  - 背诵
  - 释义自测（选择题）
  - 据义默写（三选一）
  - 断句自测（点击设置换行位置）
  - 字词识记（勾选关键词）
- 家长参考答案面板（仅家长登录可见）
- 进度本地保存（浏览器 localStorage）
- 列表筛选（年级、学期、状态），默认三年级下册

## 数据与教材
- 使用统编版小学语文三、四年级古诗词篇目，按“年级+上下册”组织
- 数据位置：见应用脚本 [app.js](file:///Users/chenaijiao/src/projects/kefan_study/app.js)

## 使用方式
- 本地直接双击打开页面入口 [index.html](file:///Users/chenaijiao/src/projects/kefan_study/index.html)
- 或开启本地静态服务器后访问（可选）
- 顶部显示总进度，左侧可筛选年级/学期/状态；默认选择三年级下册

## 登录与权限
- 学生：选择“学生”，点击“登录”，可学习但无法查看参考答案
- 家长：选择“家长”，输入密码后“登录”，可显示/隐藏参考答案
- 家长默认密码：888888（不在界面明示）
- 修改方式：在 [app.js](file:///Users/chenaijiao/src/projects/kefan_study/app.js) 中搜索 888888 并替换为你自己的密码

## 过关规则
- 必须完成以下五个模块：
  - 背诵
  - 释义自测
  - 据义默写（三选一全对）
  - 断句自测（断句位置全对）
  - 字词识记（至少命中两个本诗关键词，且不误选）
- 完成后点击“本诗过关”，自动解锁下一首

## 家长参考答案
- 释义自测：显示每题正确选项
- 据义默写：显示本次生成空格的正确字符序列
- 断句自测：显示原诗正确分行
- 字词识记：显示本诗关键词
- 入口位置：页面顶部“显示/隐藏参考答案”按钮（家长登录后可见）

## 安全与隐私
- 家长密码不在界面明示，默认值仅为初始设置，请自行更改
- 所有学习进度与登录角色保存在浏览器本地，不上传任何数据

## 常见问题
- 默认筛选未选中三年级下册？
  - 已在初始化时设为“年级：三年级、学期：下册”
- 看不到参考答案？
  - 请家长角色登录后点击顶部“显示/隐藏参考答案”
- 据义默写为什么是选择题？
  - 为降低小朋友输入成本与误差，采用三选一形式；如需填空模式可在后续版本启用
